The `download` command downloads all dependencies for the current project.

```bash
nubo download
```

Short alias:

```bash
nubo d
```

## Usage

```bash
nubo download
nubo d
```

## Example

```bash
nubo download
```

Use this after cloning a project or after editing package dependencies.
