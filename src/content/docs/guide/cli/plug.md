The `plug` command scaffolds a Nubo plugin backend.

```bash
nubo plug <folder>
```

## Usage

```bash
nubo plug <folder>
```

## Example

```bash
nubo plug backend
```

This creates the plugin backend scaffold in the given folder.

## Missing Arguments

If no folder is provided, Nubo prints:

```txt
plug requires a directory where it will create the backend
```
