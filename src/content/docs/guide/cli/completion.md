The `completion` command generates an autocompletion script for a shell.

```bash
nubo completion
```

This command is provided by the CLI framework and appears in the root help output.

## Usage

```bash
nubo completion [shell]
```

Use command help to see the supported shells and exact installation instructions.

```bash
nubo completion --help
```

Common shells usually include options such as Bash, Zsh, Fish, and PowerShell, depending on the CLI build.
